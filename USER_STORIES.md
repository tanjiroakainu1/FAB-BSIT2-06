# Food Ordering Hermanas – User Stories

| No. | As a [User] | I want a [feature] | So that [Benefits] |
|-----|-------------|---------------------|--------------------|
| 1 | Customer | I want to browse the menu grouped by categories | So that I can easily find dishes and place my order. |
| 2 | Customer | I want to add items to a cart with smooth quantity adjustments | So that I can manage my order without the page jumping or scrolling. |
| 3 | Customer | I want to choose Cash or GCash and Delivery or Pickup at checkout | So that I can pay and receive my order in the way that suits me. |
| 4 | Customer | I want to track my order status (e.g. pending, preparing, ready, delivered) | So that I know when my food is being prepared or will arrive. |
| 5 | Kitchen | I want a dashboard to view orders and update kitchen status (pending / completed) | So that I can confirm when orders are ready for delivery or pickup. |
| 6 | Delivery | I want a dedicated dashboard showing orders with delivery details and contact | So that I can deliver orders to the correct customer efficiently. |
| 7 | Delivery | I want to update order status to “Out for delivery” or “Delivered” | So that the system reflects the real status and the customer can track their order. |
| 8 | Admin | I want a dashboard showing statistics (orders today, products count, pending orders) | So that I can monitor the business and prioritize work. |
| 9 | Admin | I want to add, edit, and manage categories and products (including availability) in real time | So that the menu shown to customers is always accurate. |
| 10 | Admin | I want to view all orders and manage order, payment, and delivery status | So that I can run operations and keep customers informed. |
| 11 | Admin | I want to add Kitchen and Delivery users and view them in one place | So that I can control who can access staff and delivery dashboards. |
